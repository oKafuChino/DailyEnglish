# Database migrations

Alembic reads `DATABASE_URL` from `.env` through `app.config.Settings`.

Apply all migrations:

```bash
alembic upgrade head
```

Create a migration after changing SQLAlchemy models:

```bash
alembic revision --autogenerate -m "describe the change"
```

Review every autogenerated migration before applying it. Downgrade one revision with:

```bash
alembic downgrade -1
```

The initial migration targets PostgreSQL and uses UUID and JSONB columns. SQLite is suitable
for lightweight unit tests, but it is not the production database contract.
